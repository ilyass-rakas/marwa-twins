Place your logo image(s) in `public/` so they are served at the root.

Recommended files:
- `public/logo.png` — main logo used in the header and favicon
- `public/favicon.ico` — optional browser favicon (fallback)

Tips:
- Use a PNG (transparent background) sized around 512×512 or 1024×1024 for best results.
- If you have an SVG logo, use `logo.svg` for crisp scaling.
- After adding the files, restart the dev server or refresh the page.
